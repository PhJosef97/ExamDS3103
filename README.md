## Formula 1 Application

This project consists of two separate parts:

- `Formula1` – Frontend (client)
- `Formula1API` – Backend (ASP.NET Core Web API)

Both applications must be running at the same time for the system to work correctly.

---

## ✅ Prerequisites

Make sure you have the following installed:

- ✅ Node.js  
- ✅ .NET 7  
  Download: https://dotnet.microsoft.com/en-us/download/dotnet/7.0

---

## ▶️ How to Run the Application

### 1. Start the Frontend (`Formula1`)

Open a terminal in the `Formula1` folder and run:

```bash
npm install
npm run dev
```
Open a second terminal in the Formula1API folder and run:
```bash
dotnet watch run
```
